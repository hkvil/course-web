What is CSS Conflict?

In CSS there some rule that make html decide what CSS declaration used if theres more than one CSS Modification refer to an element.

  1. Origin

    2. Conflict
     Last Declaration win,if u declare css p element inside html with background color:red,and the element it self contain inline style with background color:yellow,than the last declaration is inline style.
    No Conflict
     When theres no conflict than it will merge,example p have style background color,and p inline have style font size.it will merge.

  2. Inheritance

    3. Any element that wrapped/child of other element will inherit its parent css style

  3. Merge

    4. Origin without conflict.

  4. Specificity
