Selector

Used to modify html element,theres 3 types of selector.CSS declaration can have multiple element with same attribute,Declared by coma.

Element

example:p{
color:blue;
}

Class

.blue{
color:blue;
}
<p class="blue">...</p>

ID

#blue{
color:blue;
}
<p id="blue">...</p>

Combining Selector

In object oriented programming theres is called dot reference,like this-->ObjectName.MethodName
In html we can use like this-->Element.class
Theres other way to combine selector,Child Selector,what it does is you pick element(child) that wrapped on other element(parent),only direct hierarchy like this-->Parent element > Child element
Descendant Selector,unlike Child you can use it even it not direct reference
Parent element child element
article p{
}
<article>..div..p../p../div...</article>
